Superstore Sales Analysis – Power BI Dashboard
Project Overview
This project is a Power BI dashboard created using the Superstore dataset. The dashboard analyzes sales performance, profit, quantity, and customer trends to help in business decision-making.
Objectives
Analyze total sales, total profit, and total quantity
Identify top-performing categories and sub-categories
Analyze sales by region and segment
Track monthly sales and profit trends
Provide business insights using interactive visuals
Tools Used
Power BI
Excel (for dataset)
DAX (Data Analysis Expressions)
Data Visualization
Dashboard Features
KPI Cards (Total Sales, Total Profit, Total Quantity)
Sales by Region (Map/Bar Chart)
Sales by Category and Sub-Category
Monthly Sales Trend (Line Chart)
Segment-wise Sales Analysis
Interactive Filters (Region, Category, Segment)
Key Insights
The West region has the highest sales.
Technology category generates the highest profit.
Some sub-categories have high sales but low profit.
Sales increase during the end of the year (Nov–Dec).
